I also love virtual reality. This website is a virtual interface that makes API calls to Getty Images, and displays the top result for three different keywords.

  • Three buttons on the bottom select a different keyword to search images for
  • Three buttons to the left select a virtual environment
  • Three buttons to the right select a color scheme for the virtual interface
It's made with A-Frame, a VR framework built on top of Three.js. Press buttons by hovering the small circle over them and mouse-clicking.
